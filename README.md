# Applet

### Running locally

Install Bun using one of the following methods:

**macOS & Linux:**
```bash
curl -fsSL https://bun.com/install | bash
```

**Package Managers:**
```bash
npm install -g bun  # or
brew install oven-sh/bun/bun
```

Verify installation:
```bash
bun --version
```

Run `bun dev` to start the app locally.
